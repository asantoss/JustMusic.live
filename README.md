# Getting Started 

These instructions will get you a copy of the project up and running on your local machine.

# Prerequisites

```
1. Node JS
2. React
3. Redux
4. Postgres
5. Sequelize 
```

# Installing

1. Clone repository
```
npm install

```

2. cd into the backend file and run node index
3. cd into the frontend file  and run npm start
